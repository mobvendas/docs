Introdução

Este documento descreve o layout de importação dos cadastros no formato TXT, e a exportação das vendas também no formato TXT através da API por intermédio do Sincronizador MobVendas. A finalidade é prover integração do sistema do parceiro com o sistema MobVendas.

Estrutura dos Arquivos

Todos os arquivos que irão interagir com o Sincronizador MobVendas deverão estar no formato TXT utilizando codificação UTF-8, com os campos separados por pipe (" | ").

Exportações MobVendas

O sincronizador mobvendas exporta os pedidos realizados através dos smartphones e através da web. Os pedidos serão separados por pastas numeradas com o código do vendedor.
